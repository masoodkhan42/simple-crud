Django REST Framework (DRF) that allows administrators to perform CRUD (Create, Read, Update, Delete) operations on student records.

🚀 Features
Create student records
View student details
Update student information
Delete student records
RESTful API architecture
Backend validation using DRF serializers
Database integration with SQLite
Django Admin Panel support
🛠️ Technologies Used
Python
Django
Django REST Framework (DRF)
SQLite
📚 CRUD Operations
Create

Add new student records to the database.

Read

Retrieve and display student information.

Update

Modify existing student records.

Delete

Remove student records from the database.

⚙️ Installation
Clone the Repository
git clone <repository-url>
cd school-management-system
Create a Virtual Environment
python -m venv venv
Activate the Virtual Environment

Windows

venv\Scripts\activate
Install Dependencies
pip install -r requirements.txt
Run Migrations
python manage.py makemigrations
python manage.py migrate
Start the Development Server
python manage.py runserver
